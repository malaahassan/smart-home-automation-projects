# Smart Home Automation Projects

A few projects I built while experimenting with ESP8266, ESP32 and Arduino boards.

Things I made:
- Fan controller with speed control
- Light Controller with Motion Sensor
- Door Soleniod Controller with RFID
- OTA update experiments

Things I used:
- ESP8266 / ESP32 / Arduino Boards
- LEDs, Resistors, Diodes, and other basic components
- Photocouplers
- Relays
- Arduino Framework (C++)
- Sinric Pro (Cloud)
- Blynk (Cloud)


These are hobby projects intended for learning and experimentation.
